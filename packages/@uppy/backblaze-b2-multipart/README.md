# @uppy/backblaze-b2-multipart

## Example
```js
const Uppy = require('@uppy/core')
const BackblazeB2 = require('@uppy/backblaze-b2-multipart')

const uppy = Uppy()
uppy.use(BackblazeB2, {

})
```

## Installation

```bash
$ npm install @uppy/backblaze-b2-multipart --save
```

Bucket: mybucket
